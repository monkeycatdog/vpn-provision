# Mihomo Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Xray + system-level OpenVPN with mihomo on both server and client. Mihomo provides Hysteria2 inbound, native OpenVPN outbound for corp split-tunnel, and a `fallback` proxy-group across multiple Shadowsocks/Outline endpoints + a `freedom` direct outbound as last-resort. Eliminates the single-Outline outage that currently kills global traffic at midday.

**Architecture:**
- **Server (VPS):** single mihomo container. Hysteria2 listener on UDP 443 (TLS, self-signed). OpenVPN outbound (mihomo-native) connects to corporate VPN — no more `openvpn-client@corporate` systemd unit, no `tun0` host iface. Two+ Shadowsocks outbounds (Outline endpoints) + `DIRECT` grouped under one `fallback` proxy-group with HTTP `url-test` health probe. Routing rules send corp IPs/domains → openvpn outbound; `geoip:ru` + RU geosite categories → DIRECT (VPS native egress); everything else → fallback group.
- **Client (laptop/phone):** mihomo (or Clash.Meta compatible client: Clash Verge Rev, Hiddify, ClashMi) speaking Hysteria2 to the server. Client config is generated by render scripts and printed as a YAML blob or `clash://` import URI.
- **State + provisioning:** keep `state/<host>/` layout, but `node.json` schema gains `outline: []` (array), `hysteria2: {password, cert_fp}`, drops `reality_*`. Render path swaps Python `string.Template` over JSON template for a YAML template using the same `Template.substitute` pattern.

**Tech Stack:** mihomo (Clash.Meta kernel), Docker, Docker Compose, bash, python3 (`string.Template`, `yaml` via PyYAML), `just`, OpenVPN (consumed by mihomo as outbound — no host install), Loyalsoldier `geoip.dat` / `geosite.dat`, systemd timer for daily geo refresh.

**Cutover strategy:** parallel deploy. Stand up mihomo on UDP 443 (Hysteria2) alongside existing Xray on TCP 443 (different protocols, same port-number OK because UDP vs TCP). When green, flip clients to mihomo URI, then remove Xray.

## File Structure

**New files:**
- `config/mihomo_config.template.yaml` — mihomo config template with `$VAR` placeholders rendered by python `string.Template`.
- `config/mihomo_client.template.yaml` — minimal client-side mihomo YAML template (one Hysteria2 proxy, default rules).
- `scripts/render_mihomo.py` — single Python entry point that takes `node.json` + `clients.json` + corp routes/domains and writes `rendered-config.yaml`. Extracted from bash heredocs so it can be unit-tested.
- `scripts/render_mihomo_client.py` — render per-client YAML.
- `scripts/tests/test_render_mihomo.py` — pytest unit tests around `render_mihomo.py`.
- `scripts/tests/test_render_mihomo_client.py` — pytest unit tests for client render.
- `scripts/tests/fixtures/node.mihomo.json` — sample state fixture.
- `scripts/tests/fixtures/clients.mihomo.json` — sample clients fixture.

**Modified files:**
- `config/xray_config.template.json` — DELETED at end of plan.
- `scripts/provision_remote.sh` — swap Xray bring-up + system OpenVPN steps for mihomo container bring-up. Generate Hysteria2 self-signed cert. Accept `--outline-uri` multiple times. Drop Reality keypair generation.
- `scripts/manage_inbound.sh` — render mihomo YAML instead of Xray JSON. `print-uri` returns a YAML/`clash://` URI instead of `vless://`. UUID-keyed clients become `username:password` pairs in Hysteria2 `users` map.
- `scripts/remote_apply_node.sh` — replace `tristate-xray` container/compose with `tristate-mihomo`. Drop OpenVPN systemd unit setup. Adjust geo asset path (mihomo default `${HOME}/.config/mihomo/` mounted to `${INSTALL_DIR}/mihomo/`).
- `scripts/diagnose_relay.sh` — probe **each** Outline endpoint in `node.json.outline[]`; replace `docker logs tristate-xray` with `tristate-mihomo`; add `curl http://127.0.0.1:9090/proxies` (mihomo REST API) to dump live proxy-group state.
- `scripts/trace_routing.sh` — adapt to mihomo's `/proxies` and `/connections` REST endpoints to show which outbound a given destination would take.
- `justfile` — recipe names stay (`just provision`, `just connection`, etc.); internal command lines change.
- `README.md` — update architecture diagram, terminology, failure-modes section.
- `.env.example` — `TRISTATE_OUTLINE_URI` → `TRISTATE_OUTLINE_URIS` (comma-separated). Add `TRISTATE_MIHOMO_DOMAIN` (optional, for Let's Encrypt later).

**Decisions locked in this plan:**
- Hysteria2 cert = **self-signed**, client uses `skip-cert-verify: true`. Reason: no domain required; quickest cutover. Future task can add ACME via a domain.
- mihomo runs as Docker container, not host binary. Reason: consistent with current Xray-via-Docker pattern.
- OpenVPN credentials (`.ovpn` + `.auth`) are parsed at render-time into `proxies[openvpn-corp]` fields (`ca`, `tls-crypt`, `username`, `password`, `server`, `port`). The host-level `openvpn-client@corporate` systemd unit is removed.
- Reality is dropped. Hysteria2 only. Client URIs change. Old `vless://` URIs stop working at cutover.
- API port `9090` listens on `127.0.0.1` only. Exposed via SSH tunnel for diagnostics.

```python
#!/usr/bin/env python3
"""Render the mihomo server config from node.json + clients.json + corp routes/domains."""
from __future__ import annotations
import argparse, ipaddress, json, socket, sys
from string import Template

import yaml


# Mirror of the Xray-template RU rule set. Keep in lock-step with the
# corresponding lists in (the now-deleted) config/xray_config.template.json
# when porting changes.
RU_GEOSITES = [
    "2gis", "category-bank-ru", "category-betting-ru", "category-ecommerce-ru",
    "category-entertainment-ru", "category-gov-ru", "category-media-ru",
    "category-medicine-ru", "category-retail-ru", "category-travel-ru",
    "kinopoisk", "mailru", "mailru-group", "myoffice-ru", "ok",
    "category-ru", "sber", "vk", "yandex",
]

RU_DOMAINS = [
    "alfabank.ru", "avito.ru", "beeline.ru", "cdek.ru", "detmir.ru",
    "dns-shop.ru", "eldorado.ru", "gazprombank.ru", "gosuslugi.ru", "hh.ru",
    "ivi.ru", "lamoda.ru", "megafon.ru", "mts.ru", "more.tv", "mvideo.ru",
    "okko.tv", "ozon.ru", "pochta.ru", "qiwi.com", "raiffeisen.ru",
    "rostelecom.ru", "rutube.ru", "samokat.ru", "superjob.ru", "tbank.ru",
    "tele2.ru", "tinkoff.ru", "vtb.ru", "wildberries.ru",
]


def load_routes(path):
    cidrs = []
    if not path:
        return cidrs
    for line in open(path):
        s = line.strip()
        if not s or s.startswith("#") or not s.startswith("route "):
            continue
        _, addr, mask = s.split()
        net = ipaddress.IPv4Network(f"{addr}/{mask}", strict=False)
        cidrs.append(str(net))
    return cidrs


def load_domains(path):
    domains = []
    if not path:
        return domains
    for line in open(path):
        s = line.strip()
        if not s or s.startswith("#"):
            continue
        domains.append(s)
    return domains


def resolve_to_cidrs(host):
    """Return CIDR set for host (literal IP or DNS name). Mirrors the
    manage_inbound.sh loop-guard logic so SS outbound traffic to Outline
    and VPS-self traffic never re-enters the proxy."""
    out = set()
    if not host:
        return out
    try:
        ipaddress.ip_address(host)
        out.add(f"{host}/32" if ":" not in host else f"{host}/128")
        return out
    except ValueError:
        pass
    try:
        for info in socket.getaddrinfo(host, None):
            ip = info[4][0]
            try:
                addr = ipaddress.ip_address(ip)
                out.add(f"{ip}/128" if addr.version == 6 else f"{ip}/32")
            except ValueError:
                continue
    except OSError:
        pass
    return out


def build_loop_guard(node):
    guard = set()
    for ep in node["outline"]:
        guard |= resolve_to_cidrs(ep.get("address"))
    guard |= resolve_to_cidrs(node.get("host"))
    if not guard:
        guard.add("240.0.0.0/32")  # documentation range; never matches
    return sorted(guard)


def build_proxies(node):
    proxies = []
    for ep in node["outline"]:
        proxies.append({
            "name":     f"ss-{ep['name']}",
            "type":     "ss",
            "server":   ep["address"],
            "port":     ep["port"],
            "cipher":   ep["method"],
            "password": ep["password"],
            "udp":      True,
        })
    proxies.append({"name": "direct-out", "type": "direct"})
    ovpn = node["openvpn_corp"]
    block = {
        "name":   "ovpn-corp",
        "type":   "openvpn",
        "server": ovpn["server"],
        "port":   ovpn["port"],
        "proto":  ovpn["proto"],
        "udp":    ovpn["proto"] == "udp",
        "ca":     ovpn["ca_pem"],
    }
    if "username" in ovpn:
        block["username"] = ovpn["username"]
        block["password"] = ovpn["password"]
    if "tls_crypt_pem" in ovpn:
        block["tls-crypt"] = ovpn["tls_crypt_pem"]
    if "cert_pem" in ovpn:
        block["cert"] = ovpn["cert_pem"]
        block["key"]  = ovpn["key_pem"]
    if "cipher" in ovpn:
        block["cipher"] = ovpn["cipher"]
    if "auth" in ovpn:
        block["auth"] = ovpn["auth"]
    proxies.append(block)
    return proxies


def build_rules(loop_guard, routes, corp_domains):
    rules = []
    # 1) Loop guard FIRST — outline endpoints + VPS-self always direct,
    #    so SS outbound packets never re-enter Hysteria2 inbound.
    for cidr in loop_guard:
        rules.append(f"IP-CIDR,{cidr},direct-out,no-resolve")
    # 2) Corporate domains → OpenVPN outbound
    for d in corp_domains:
        rules.append(f"DOMAIN-SUFFIX,{d},ovpn-corp")
    # 3) Corporate IP CIDRs → OpenVPN outbound
    for cidr in routes:
        rules.append(f"IP-CIDR,{cidr},ovpn-corp,no-resolve")
    # 4) Russian geosite categories → direct (VPS native egress)
    for site in RU_GEOSITES:
        rules.append(f"GEOSITE,{site},direct-out")
    # 5) Explicit RU domain suffixes → direct
    for d in RU_DOMAINS:
        rules.append(f"DOMAIN-SUFFIX,{d},direct-out")
    # 6) RU geoip → direct
    rules.append("GEOIP,RU,direct-out,no-resolve")
    # 7) Default → fallback group (Outline endpoints + direct last resort)
    rules.append("MATCH,GLOBAL")
    return rules


def main():
    p = argparse.ArgumentParser()
    p.add_argument("--template", required=True)
    p.add_argument("--node",     required=True)
    p.add_argument("--clients",  required=True)
    p.add_argument("--routes",   default="")
    p.add_argument("--corp-domains", default="")
    p.add_argument("--output",   required=True)
    args = p.parse_args()

    node    = json.load(open(args.node))
    clients = json.load(open(args.clients))
    routes  = load_routes(args.routes)
    corp_domains = load_domains(args.corp_domains)
    loop_guard = build_loop_guard(node)

    hy2_users = {c["email"]: c["password"] for c in clients}
    proxies   = build_proxies(node)
    global_proxies = [p["name"] for p in proxies if p["name"].startswith("ss-")] + ["direct-out"]
    rules     = build_rules(loop_guard, routes, corp_domains)

    tmpl = Template(open(args.template).read())
    rendered = tmpl.substitute(
        LISTEN_PORT=node["listen_port"],
        HY2_USERS=json.dumps(hy2_users),
        PROXIES=json.dumps(proxies),
        GLOBAL_PROXIES=json.dumps(global_proxies),
        RULES=json.dumps(rules),
    )

    # Round-trip through YAML parser so output is canonical YAML, not the
    # JSON-blob substitution form that string.Template produced.
    obj = yaml.safe_load(rendered)
    with open(args.output, "w") as fh:
        yaml.safe_dump(obj, fh, sort_keys=False, allow_unicode=True)


if __name__ == "__main__":
    main()
```

```python
    raw = open(path).read()
    out = {}
    # remote <server> <port> <proto>
    for line in raw.splitlines():
        s = line.strip()
        if s.startswith("remote "):
            parts = s.split()
            out["server"] = parts[1]
            out["port"]   = int(parts[2]) if len(parts) > 2 else 1194
        elif s.startswith("proto "):
            out["proto"] = s.split()[1]
        elif s.startswith("cipher "):
            out["cipher"] = s.split()[1]
        elif s.startswith("auth "):
            out["auth"] = s.split()[1]

        import re
        m = re.search(rf"<{tag}>\s*(.*?)\s*</{tag}>", raw, re.S)
        return m.group(1).strip() if m else None

    if (tc := extract("tls-crypt")): out["tls_crypt_pem"] = tc + "\n"
    if (cert := extract("cert")):   out["cert_pem"] = cert + "\n"
    if (key := extract("key")):     out["key_pem"]  = key + "\n"
    out.setdefault("proto", "udp")
```
